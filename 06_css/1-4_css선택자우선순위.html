선택자 우선순위

기본적으로 css 속성은 위에서부터 아래로 적용이 되지만, 같은 태그에 여러 개의 css속성이 설정된 경우에는 우선순위에 따라 스타일이 적용된다. 

        (우선순위 낮음)
        ┃
        ┃ 태그 선택자 : p { color: blue; }
        ┃      ∨
        ┃ 클래스 선택자 : class 속성을 가진 요소를 선택 .highlight { background-color: yellow; }
        ┃      ∨
        ┃ 아이디 선택자 : #header { font-size: 24px; }
        ┃      ∨
        ┃ 인라인 선택자 : HTML 태그 내부에 직접 style 속성을 사용하여 CSS 스타일을 적용하는 것을 말합니다.
        ┃                유지보수와 가독성 측면에서 추천되지 않습니다. 스타일은 일반적으로 style태그나 외부 CSS 파일에서 관리하는 것이 좋습니다.
        ┃      ∨
        ┃ !important 
        ↓
        (우선순위 높음)
        순서로 우선순위를 가진다.
