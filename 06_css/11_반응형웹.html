반응형 웹 디자인

1. 반응형 웹의 핵심 구성 요소

1-1. 뷰포트(Viewport)

<meta name="viewport" content="width=device-width, initial-scale=1.0">

- 모바일 기기에서 웹페이지가 표시되는 영역을 설정

- 기기의 실제 화면 크기에 맞춰 컨텐츠를 표시

1-2. 미디어 쿼리(Media Queries)

화면 크기에 따라 다른 스타일을 적용할 수 있게 해주는 CSS3 기능

/* 모바일 우선 설계 */
/* 기본 스타일 (모바일) */
.container { width: 100%; }

/* 태블릿 */
@media (min-width: 768px) {
	.container { width: 750px; }
}

/* 데스크톱 */
@media (min-width: 1024px) {
	.container { width: 970px; }
}

3. 반응형 디자인 패턴

  • Mostly Fluid: 작은 화면에서는 수직으로 쌓이고, 큰 화면에서는 격자로 배치
  • Column Drop: 화면이 작아지면서 컬럼이 순차적으로 아래로 떨어짐
  • Layout Shifter: 화면 크기에 따라 레이아웃이 완전히 다르게 변경
  • Tiny Tweaks: 작은 변화만 주는 단순한 패턴
  • Off Canvas: 작은 화면에서 콘텐츠를 숨겼다가 필요할 때 보여줌
